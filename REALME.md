# Simple Card CSS2

A basic card layout created using HTML and CSS2 only.

## 🔍 Features

- Float-based layout (no Flexbox or Grid)
- Clean design with image and text
- Custom mini typography styles

## 🛠️ Technologies

- HTML5
- CSS2 (float, clear, inline-block)

## 📁 Files

- `index.html`
- `style.css`
- `normalize.css`
- `1.jpg`
